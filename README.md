# Renuga Sree | AI/ML Engineer Portfolio

A production-grade, high-impact portfolio built to showcase engineering depth in AI, Data Science, and Scalable Systems.

## 🚀 Technical Stack

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** Vercel / Docker-ready

## 🛠️ Key Features

- **Adaptive Case Study:** Deep dive into the "Adaptive AI Learning Assistant" with architecture diagrams and custom mastery logic visualization.
- **Engineering Showcase:** Structured project cards detailing Problem → Solution → Tech → Outcome.
- **Impact-Driven Experience:** Professional history focused on outcomes and system-level contributions.
- **Responsive Design:** Premium dark theme with smooth transitions and glassmorphism.

## 🏃 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Project Structure

```text
src/
├── app/            # Next.js App Router
├── components/     # Reusable UI components
├── styles/         # Design system & globals
└── public/         # Static assets
```
