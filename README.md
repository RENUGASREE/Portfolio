# Renuga Sree | AI/ML Engineer Portfolio

A premium, production-grade technical product showcase. Designed for high-impact visibility in Data Science and AI/ML Engineering roles.

## 🎯 The Objective
This portfolio is engineered to demonstrate **production exposure**. It showcases end-to-end system thinking, API design, and scalable deployment practices, positioning the candidate as a junior engineer with production experience.

## 🚀 Technical Architecture

- **Core:** [Next.js 15](https://nextjs.org/) (App Router)
- **Engine:** [TypeScript](https://www.typescriptlang.org/) for type-safe system logic
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) with a custom Premium Glassmorphism design system
- **Interactions:** [Framer Motion](https://www.framer.com/motion/) for high-fidelity animations
- **Infrastructure:** Vercel (Recommended) or GitHub Pages

## 🛠️ Key Product Features

- **Premium Navigation:** Centered capsule design with smooth scroll spy and a high-fidelity fullscreen mobile overlay.
- **Elite Case Study:** Deep dive into the *Adaptive AI Learning Assistant* with production system metrics and mastery logic visualization.
- **Engineering Showcase:** Product-focused project cards using the *Problem → Solution → Tech → Outcome* framework.
- **Unified Branding:** Custom "RS" geometric logo reflecting technical depth and minimal luxury.
- **Robust Downloads:** Reliable PDF resume delivery system optimized for all devices.

## 🏃 Engineering Setup

### 1. Installation
```bash
npm install
```

### 2. Local Development
```bash
npm run dev
```

### 3. Production Build
```bash
npm run build
```

## 📦 Deployment Strategy

### Vercel (Recommended)
Simply connect your GitHub repository to Vercel. The project is pre-configured for zero-config Vercel deployment.

### GitHub Pages
1. Go to `Settings > Pages` in your repository.
2. Set the Source to **GitHub Actions**.
3. Pushing to `main` will automatically trigger the deployment pipeline.
